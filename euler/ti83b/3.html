Largest Prime Factor (TI-83 BASIC)

Checks the factors of 600851475143 with trial division. We find the factors in ascending order, and divide them out of the searched number as we go. The remnant only then has prime factors not less than the ones already found. Since finding a factor greater than the square root of the remnant would imply that there is also one smaller than the square root of the remnant, we can limit the search to not exceed the square root of the remnant, since we have already found all smaller factors.

Using the following facts, we can speed up the program by memoizing previous factors.

  • The smallest factor of a natural number is prime
  • If a factor is not divisible by any previous factors, then it is prime
